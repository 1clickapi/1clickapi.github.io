    Query supported API on 1Click.io service

    A sample usage of querying 1Click.io for supported REST api curl 

    
curl https://oneklikstreet.com/api/v1/?format=json -v --user "username:api_key" --digest

    Query all the conference id's created by a user name

    A sample usage of querying 1Click.io for supported REST api curl 

    
curl https://oneklikstreet.com/api/v1/conference/?format=json -v --user "username:api_key" --digest
